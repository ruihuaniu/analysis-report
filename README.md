# Transaction Analysis Report

> A beautiful, interactive spending dashboard built entirely in a single HTML file.

---

## 🌟 Project Snapshot

This project is a polished financial analysis report for a year of Mastercard transactions. It renders a clean, magazine-style dashboard with:

- summary metrics
- monthly spend trends
- category breakdowns with expandable transaction details
- top merchant insights
- observations and spending recommendations
- a handy upload flow for CSV / Excel imports

---

## 🎯 Why it’s useful

This dashboard is perfect for:

- visualizing personal or small-business spending patterns
- spotting high-cost categories and recurring outgoings
- reviewing merchant behavior and monthly trends
- exploring a self-contained demo without a backend

---

## 🚀 Features

- responsive layout with modern typography and cards
- Chart.js-powered visualizations
- interactive category drilldown with transaction panels
- merchant search, sort, and pagination controls
- drag-and-drop / click upload support
- column mapping modal for flexible import handling

---

## 🧩 What’s included

- `index.html` — complete app, styling, sample data, and logic all in one file

---

## 🧪 Quick start

1. Open `index.html` in a modern browser.
2. Explore the built-in report sections:
   - Summary
   - Monthly Spending
   - Category Breakdown
   - Top Merchants
   - Key Observations
   - Areas to Review
3. Upload your own `.csv`, `.xlsx`, or `.xls` statement export.
4. Map columns if prompted, then view the generated insights.
